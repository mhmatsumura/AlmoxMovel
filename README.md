# Almox Móvel (ANDROID)
#### Aplicativo desenvolvido para auxiliar meus colegas de trabalho a efetuarem requisições de materiais no almoxarifado de forma simplificada e ágil.

##### Tecnologias utilizadas:
- itextpdf
- AsyncTask
- Intent
- emailIntent
- SharedPreferences
- Uri
- AppCompatActivity
- FileProvider
- BufferedWriter
- FileOutputStream
- OutputStreamWriter
- SimpleDateFormat
- Writer
- Date
- Log
- AlertDialog
- Toast
- Spinner
- TextView
- ImageButton
- ArrayAdapter
- AdapterView
- Menu
- Bundle
